# Retrieval Benchmark: Strategy A (Raw) vs Strategy B (AI-Enhanced)

Generated by `python -m rag.benchmarks`.

* **Strategy A (Raw Vector Search)** -- embed the user query verbatim and run cosine top-k against the ChromaDB collection.
* **Strategy B (AI-Enhanced Retrieval)** -- send the query through the mocked `vertexai.generative_models.GenerativeModel` (Gemini 1.5 Flash analogue) for rewriting/expansion, then embed and search.

All vectors are produced by the mocked `vertexai.language_models.TextEmbeddingModel` (`textembedding-gecko@003`), routed locally to `sentence-transformers/all-MiniLM-L6-v2`.
